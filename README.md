#  Processamento da Informação (PI) - UFABC

Este repositório contém as atividades e projetos desenvolvidos durante a disciplina de **Processamento da Informação** no quadrimestre de 2026 na **Universidade Federal do ABC**.

O foco principal é o aprendizado de lógica de programação utilizando a linguagem **Python**.

##  Estrutura do Repositório

O conteúdo está organizado cronologicamente conforme o progresso das aulas:

*   **01 - Estruturas de repetição**: Exercícios com laços `while`, `for` e lógica de números primos.
*   **02 - Strings**: Manipulação de textos e cadeias de caracteres.
*   **03 - Vetores**: Trabalho com listas (arrays), médias e filtragem de dados.
*   **04 - Matrizes**: Operações matemáticas e lógica em estruturas multidimensionais.

##  Tecnologias Utilizadas
*   Python 3.x
*   Lógica de Programação e Algoritmos
